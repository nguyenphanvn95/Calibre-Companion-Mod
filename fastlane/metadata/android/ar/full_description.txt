تطبيق مصاحب غير رسمي لـ Calibre Web و Calibre Web Automated يتيح لك تصفح مجموعتك من الكتب وإدارة مكتبتك الرقمية أثناء التنقل.

تم تصميمه باستخدام <b>Flutter</b> و <b>Material You</b> لتوفير تجربة مستخدم حديثة وبديهية وسريعة الاستجابة.

<b>الميزات المدعومة:</b>
<ul>
    <li>تصفح مكتبة Calibre Web الخاصة بك</li>
    <ul>
        <li>عرض الكتب حسب المؤلف والسلسلة والاتجاهات وغيرها</li>
        <li>عرض معلومات مفصلة عن الكتب</li>
        <li>الوصول إلى إحصائيات شاملة عن المجموعة</li>
    </ul>
    <li>إدارة المكتبة</li>
    <ul>
        <li>وضع علامة على الكتب كمقروءة أو غير مقروءة</li>
        <li>أرشفة الكتب وإلغاء أرشفتها</li>
        <li>تنظيم الكتب في أرفف</li>
        <li>إدارة أرففك وتخصيصها</li>
    </ul>
    <li>القراءة والأجهزة</li>
    <ul>
        <li>تنزيل الكتب مباشرةً على جهازك</li>
        <li>إرسال الكتب إلى قارئك الإلكتروني (Kindle/Kobo) عبر send2ereader</li>
        <li>استخدام وظيفة البريد في Calibre Web لتسليم الأجهزة</li>
    </ul>
    <li>تكامل الأتمتة</li>
    <ul>
        <li>يعمل مع Calibre Web Automated</li>
        <li>تنزيل الكتب تلقائيًا باستخدام calibre-web-automated-book-downloader</li>
    </ul>
</ul>

يُرجى ملاحظة أن التطبيق قيد التطوير حاليًا. يمكنك فتح مشكلات على GitHub للإبلاغ عن الأخطاء وتقديم الملاحظات واقتراح الأفكار الخاصة بالميزات.
